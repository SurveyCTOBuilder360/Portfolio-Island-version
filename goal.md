# Project Goal

## Role
You are a React + Vite specialist with expertise in building modern, performant web applications using the latest React ecosystem tools.

## Project: 3D Animated Portfolio Website

### Overview
Building a stunning, interactive 3D animated portfolio website using React and Vite.

### Tech Stack
- **Framework**: React 18+
- **Build Tool**: Vite
- **3D Graphics**: Three.js / React Three Fiber
- **Animations**: Framer Motion / GSAP / React Spring
- **Styling**: Tailwind CSS (or alternatives)
- **Routing**: React Router DOM

### Objectives
1. Create an immersive 3D experience with smooth animations
2. Ensure optimal performance with Vite's fast build times
3. Implement responsive design for all devices
4. Showcase projects and skills in an engaging, interactive way
5. Maintain clean, modular, and scalable code architecture

### Key Features
- Interactive 3D scene with camera controls
- Smooth page transitions and scroll-based animations
- Project gallery with 3D effects
- Contact section with form validation
- Dark/light theme support (optional)
- SEO optimization
- Fast loading times (< 3 seconds)

### Success Criteria
- Website is visually impressive and memorable
- All animations run at 60fps
- Mobile-responsive across all screen sizes
- Codebase follows React best practices
- Easy to deploy and maintain
